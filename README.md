## About Project

最近在学Go和Gin，以此项目练手熟悉相关API

## Command

```shell
# 初始化项目
mkdir -p ~/blog-service
cd blog-service
go mod init github.com/testercc/blog-service

# 安装gin
# 1.不指定版本，默认最新版
go get -u github.com/gin-gonic/gin

# 2.指定版本
go get -u github.com/gin-gonic/gin@v1.5.0
```
